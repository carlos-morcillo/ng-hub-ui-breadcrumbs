## Resumen

`ng-hub-ui-breadcrumbs` genera breadcrumbs a partir de la configuración de rutas de Angular y permite personalizar plantillas, iconos y renderizado.

## Instalación

```bash
npm install ng-hub-ui-breadcrumbs
```

## Uso rápido

```typescript
import { HubBreadcrumbComponent } from 'ng-hub-ui-breadcrumbs';
```

## Características

- Integración con Angular Router.
- Breadcrumbs dinámicos y configurables.
- Soporte para plantillas y estilos personalizados.
- Útil para aplicaciones documentales, paneles y navegación compleja.
